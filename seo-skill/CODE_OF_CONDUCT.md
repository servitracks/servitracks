# Code of Conduct

## Our Pledge

We are committed to a respectful, harassment-free project environment for contributors and users.

## Expected Behavior

- Be direct, constructive, and specific.
- Assume good intent while correcting technical issues with evidence.
- Respect different experience levels and implementation constraints.
- Keep discussions focused on the project.

## Unacceptable Behavior

- Harassment, threats, insults, or discriminatory language.
- Publishing private information without permission.
- Repeated off-topic disruption.
- Bad-faith security reports or attempts to expose user secrets.

## Enforcement

Maintainers may remove comments, close issues, restrict participation, or ban contributors who violate this code of conduct. To report a problem, use the contact path in `SUPPORT.md` or `SECURITY.md` if the issue is security-sensitive.
